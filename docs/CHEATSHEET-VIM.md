# Vim Cheatsheet

## Key Mappings

Key Map | What it do 
--------|------------
, | leader key
,, | save current file
,qq | exit current file faster
,ev | edit ~/.config/nvim/init.vim
,eg | edit ~/.gitconfig
,ez | edit ~/.zshrc
,v | toggle set paste
,n | toggle NERDTree
,nb | NERDTree from Bookmark
,nf | NERDTree find
,gs | git status
,gc | git commit
,ge | :Gedit
,e | open fzf
\<space> | clear current search highlight
C-e and C-y | scroll 3x faster than normal
